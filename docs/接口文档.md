# API 接口文档 — LLM Wiki Approach

## 设计理念

采用"万物皆MD"架构：API 返回 JSON 信封，核心内容为 Markdown 字符串。知识图谱从 vault 中的 `[[wikilinks]]` 自动派生，无需独立存储图结构。

## 基础信息

- Base URL: `http://localhost:8000`
- 响应格式: JSON
- 认证: 无（MVP 阶段）

---

## 端点列表

### 健康检查

```
GET /api/health
```

响应:
```json
{"status": "ok"}
```

### 文档管理

#### 上传并解析文档

```
POST /api/documents/upload
Content-Type: multipart/form-data
```

参数: `file` (PDF/MD/TXT，最大 512MB)

响应:
```json
{
  "document_id": "doc_a1b2c3d4",
  "status": "parsed",
  "chapter_count": 12,
  "total_chars": 385000,
  "vault_paths": ["textbooks/病理学"]
}
```

副作用: 解析后的章节同时写入 `data/vault/textbooks/{title}/` 目录。

#### 上传并返回完整解析结果

```
POST /api/parse/upload
Content-Type: multipart/form-data
```

返回完整的 ParsedDocument JSON（用于前端章节预览）。

#### 列出已上传文档

```
GET /api/documents
```

响应:
```json
{
  "documents": [
    {
      "document_id": "doc_a1b2c3d4",
      "filename": "病理学.pdf",
      "title": "病理学",
      "format": "pdf",
      "size_bytes": 826000,
      "status": "parsed",
      "chapter_count": 15,
      "total_chars": 420000,
      "extraction_status": "none",
      "created_at": "2026-05-10T12:00:00Z"
    }
  ]
}
```

#### 获取文档详情

```
GET /api/documents/{document_id}
```

响应:
```json
{
  "document_id": "doc_a1b2c3d4",
  "title": "病理学",
  "format": "pdf",
  "total_chars": 420000,
  "chapters": [
    {"chapter_id": "ch_001", "title": "第一章 绪论", "char_count": 8500, "page_start": 1, "page_end": 15}
  ]
}
```

### 知识提取

#### 触发 LLM 知识点提取

```
POST /api/extraction/run
Content-Type: application/json
```

请求:
```json
{"document_id": "doc_a1b2c3d4"}
```

响应:
```json
{
  "document_id": "doc_a1b2c3d4",
  "status": "ready",
  "concepts_written": 42,
  "vault_paths": ["concepts/炎症.md", "concepts/细胞损伤.md"]
}
```

副作用: 每个知识点生成一个 Obsidian-flavored MD 文件到 `data/vault/concepts/`。

## Vault 文件格式 (Obsidian-compatible)

每个概念文件位于 `data/vault/concepts/{safe_name}.md`。完整规范以 `AGENTS.md`
的 `SSOT: Concept Vault Schema` 为准：

```markdown
---
id: concept_炎症
category: 核心概念
aliases: [inflammation, 炎性反应]
sources:
  - textbook_id: book_01
    chapter_id: ch_004
    page_start: 78
    page_end: 82
    definition: 炎症是具有血管系统的活体组织对各种损伤因子的刺激所发生的防御性反应。
    evidence: 炎症(inflammation)是具有血管系统的活体组织对各种损伤因子...
---

# 炎症

炎症是具有血管系统的活体组织对各种损伤因子的刺激所发生的防御性反应。

## 关系

- 前置依赖: [[细胞损伤]]
- 包含: [[急性炎症]], [[慢性炎症]]
- 并列: [[免疫应答]]

## 来源证据

> 炎症(inflammation)是具有血管系统的活体组织对各种损伤因子...

来源: 病理学 ch_004
```

关系类型:
| 类型 | 含义 |
|------|------|
| prerequisite | 前置依赖 |
| contains | 包含 |
| parallel | 并列 |
| applies_to | 应用于 |
