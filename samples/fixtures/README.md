# Fixtures

小型可提交样例，用于 parser 和后续 pipeline 的 red/green TDD。

- `markdown/structured_course.md`: Markdown 章节、子标题、代码块、图片引用。
- `markdown/sys1/lab2-1.md`: 系统 I 高质量原文样本，覆盖全加器、行波进位、超前进位、全加减法器。
- `markdown/sys1/lab3-1.md`: 系统 I 高质量原文样本，覆盖时序电路、有限状态机、状态编码与 FSM 设计方法。
- `text/structured_course.txt`: TXT 章标题、节标题、编号条目、普通段落。

PDF 样例暂时使用本地 `samples/textbooks/` 的赛方教材，不进入 Git。
